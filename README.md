# Реализация контейнера std::string (C++)

## Введение 📝

**Цель работы**: реализовать конструктор ```std::string``` через собственный класс ```String```

К текущему моменту в проекте имеется:

* 9 конструкторов
* 35 методов
* перегружены операции сложения (как внешние функции от двух параметров)
* 1 деструктор

Структура проекта:

```
├──📄CMakeLists.txt       # правила сборки
├──📄main.cpp             # тесты класса
├──📄string.cpp           # определение конструкторов, методов, деструкторов и внешних функций
├──📄string.hpp           # объявление класса и внешних функций
└──📄README.md
```

# Функционал класса String ⚙️

| Метод | Описание |
| :---: | :---: |
| **Конструкторы (Делегирующие) и деструктор** | |
| `String()` | Конструктор по умолчанию. Создает пустую строку (выделяет 1 байт под `\0`). |
| `String(const char* str)` | Создает строку на основе Си-строки. |
| `String(size_t n, char symbol)` | Создает строку, состоящую из `n` одинаковых символов `symbol`. |
| `String(const String& other)` | Конструктор копирования. |
| `String(const char* str, size_t n)` | Создает строку из первых `n` символов Си-строки `str`. |
| `String(const char* str, size_t start, size_t n)` | Создает строку из подстроки `str`, начиная с индекса `start` длиной `n`. |
| `String(const String& other, size_t n)` | Создает строку из первых `n` символов другой строки `other`. |
| `String(const String& other, size_t start, size_t n)` | Создает строку из подстроки `other`, начиная с индекса `start` длиной `n`. |
| `~String()` | Деструктор. Освобождает выделенную динамическую память. |
| **Операторы присваивания и сложения** | |
| `operator=(const char* str)` | Присваивает Си-строку. |
| `operator=(String other)` | Оператор копирующего присваивания (идиома Copy-and-Swap). |
| `operator+=(const char* str)` | Дописывает Си-строку в конец текущей. |
| `operator+=(const String& other)` | Дописывает другую строку `String` в конец текущей. |
| **Доступ к элементам** | |
| `operator[](size_t n)` | Доступ к символу по индексу `n` (неконстантный / константный). |
| `front()` | Возвращает ссылку на первый символ строки (неконстантный / константный). |
| `back()` | Возвращает ссылку на последний символ строки (неконстантный / константный). |
| `get_chars()` | Возвращает указатель на внутренний Си-массив символов (`const char*`). |
| **Размер и емкость** | |
| `size()` / `length()` | Возвращают логическую длину строки (количество символов). |
| `capacity()` | Возвращает полный физический размер выделенного буфера на куче (включая `\0`). |
| `max_size()` | Возвращает максимальный логический размер, который может вместить текущий буфер. |
| `empty()` | Проверяет строку на пустоту (`size == 0`). |
| **Управление емкостью и размером** | |
| `reserve(size_t n)` | Резервирует память под строку (увеличивает `capacity`), не меняя логический размер. |
| `resize(size_t n)` | Меняет размер строки до `n`. При расширении заполняет мусором, ставит `\0`. |
| `resize(size_t n, char symbol)` | Меняет размер строки до `n`. При расширении заполняет новые места символом `symbol`. |
| `shrink_to_fit()` | Ужимает физическую емкость буфера до минимально необходимой (`size + 1` байт). |
| **Управление содержимым** | |
| `push_back(char symbol)` | Добавляет один символ в конец строки (если нужно предварительно увеличивает память). |
| `pop_back()` | Удаляет последний символ строки. |
| `insert(size_t pos, const char* str)` | Вставляет Си-строку в позицию `pos`. |
| `insert(size_t pos, const String& other)` | Вставляет строку `String` в позицию `pos`. |
| `erase(size_t pos)` | Удаляет один символ в позиции `pos`. |
| `erase(size_t pos, size_t count)` | Удаляет `count` символов начиная с `pos`. |
| `clear()` | Очищает строку, сбрасывая её размер до нуля. Память не освобождается. |
| `replace(size_t pos, size_t count, const char* str)` | Заменяет кусок строки от `pos` длиной `count` на Си-строку `str`. |
| `replace(size_t pos, size_t count, const String& other)` | Заменяет кусок строки на другую строку `String`. |
| `replace(size_t start, size_t end, char old_s, char new_s)` | Заменяет все вхождения символа `old_s` на `new_s` в диапазоне от `start` до `end`. |
| `swap(String& other)` | Меняет содержимое и указатели двух объектов `String` местами. |
| **Поиск и сравнение** | |
| `find_first(char symbol)` | Ищет первое вхождение символа. Возвращает индекс или `std::nullopt`. |
| `find_last(char symbol)` | Ищет последнее вхождение символа. Возвращает индекс или `std::nullopt`. |
| `substr(size_t pos, size_t count)` | Вырезает и возвращает новую строку из текущей, начиная с `pos` длиной `count`. |
| `compare(const String& other)` | Сравнивает строку с `other` лексикографически (аналог `strcmp`). |




| Внешняя функция | Описание |
|:---------------:|:--------:|
| `operator+(const String&, const String&)` | Возвращает результат сложения двух строк. |
| `operator+(const String&, const char*)` | Сложение строки и Си-строки справа. |
| `operator+(const char*, const String&)` | Сложение Си-строки и строки слева (симметричный оператор). |
| `operator<<(std::ostream& out, const String& str)` | Выводит строку String. |

# Установка репозитория 👾

### **Чтобы скачать весь проект, нужно:**

1. В Вашем терминале необходимо указать директорию, куда вы хотите скачать данный проект.

2. Затем в терминале введите:
```
git@github.com:SergioFoma/CppString.git
```

3. Дождитесь скачивания.

4. После чего будет создана рабочая директория с названием репозитория.

# Запуск программы 🚀

1. Для запуска необходимо перейти в папку ```CppString```.
```
cd CppString
```

2. После этого для сборки проекта необходимо прописать команду:
```
cmake -S. -B Build
```

, где ```Build``` - папка, где будет создан Makefile и сопуствующие файлы.

3. Для сборки исполняемого файла необходимо прописать команду:
```
cmake --build Build
```

4. Запуск программы:
```
Build/string
```
